Color Explorer App

Color Explorer is a Flutter mobile application that allows users to explore and interact with different colors through a simple and intuitive interface. The app is designed with multiple screens, smooth navigation, and real-time search functionality.

🖥️ Screens Overview
🔐 Screen 1: Login & Registration

This screen contains three input fields for Name, Username, and Password. The user must fill all fields before logging in. A Login button navigates the user to the main app. Password visibility toggle is included for better usability.

🧭 Screen 2: Main Navigation Shell

This screen uses a Bottom Navigation Bar with two sections:

🎨 Gallery
⚙️ Settings

Users can switch between screens dynamically without losing state.

🎨 Screen 3: Color Gallery

A dashboard-style screen displaying a 2-column GridView of colored cards. Users can search colors in real-time using a search bar. Tapping a color card opens a detailed view of that color.

🌈 Screen 4: Color Detail Screen

This screen displays the selected color as the full background. A central back button allows the user to return to the gallery screen.

⚙️ Screen 5: Settings & Developer Info

Displays a welcome message, followed by a ListView showing developer name and contact email. A logout button clears navigation history and returns the user to the login screen.
